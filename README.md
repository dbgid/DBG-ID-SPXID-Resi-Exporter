# Build
```bash
git clone https://github.com/dbgid/DBG-ID-SPXID-Resi-Exporter
```
Open your android AIDE APK, choose project the folder this repo click play or build on AIDE.

# Feature
- Image OCR Recognized
- Export the result of RESI to xlsx file

# Usage
This apk may can help the courier `SPX` to report any status of RESI for daily working.
